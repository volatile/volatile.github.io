Getting started

1 Install Go

Before creating an app with Volatile, you must have a working Go installation.
Follow the official guide for this, and don't forget to set your GOPATH environment variable.

2 Install the Volatile Core

Get the Core package with the go get command: 

$ go get -u github.com/volatile/core

3 Create your app

Create a new project directory in your GOPATH and go into it: 
$ mkdir -p $GOPATH/src/$USER/volatileapp && cd $_
Open your favorite text editor and create an app.go file in this directory: 
package main

import (
    "fmt"
    "log"
    "time"

    "github.com/volatile/core"
)

func main() {
    // Log
    core.Use(func(c *core.Context) {
        start := time.Now()
        c.Next()
        log.Printf(" %s  %s  %s", c.Request.Method, c.Request.URL, time.Since(start))
    })

    // Response
    core.Use(func(c *core.Context) {
        fmt.Fprint(c.ResponseWriter, "Hello, World!")
    })

    core.Run()
}

4 Run your app

Return in your terminal, build and run your new app: 
$ go build && ./volatileapp
The server is now reachable at http://localhost:8080/.
