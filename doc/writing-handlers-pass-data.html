Pass data

To transmit data from a handler to another, the context has the Data field, which is a map[string]interface{}.
For example, it can be used to parse request data a single time in a top handler, and transmit the result trough the handlers chain without doing the work each time. 

// Set data
core.Use(func(c *core.Context) {
    c.Data["id"] = 123
})

// Read data
core.Use(func(c *core.Context) {
    println(c.Data["id"].(int))
})
