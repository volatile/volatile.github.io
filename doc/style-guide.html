Style guide

The Volatile style guide recommends best practices for writing handlers and helpers packages.

Rules

  1. Use the Go standard library as much as possible.
  2. When package receives global configuration, it must be with function Init.
  3. When package provides a handler for the default handlers stack, it must be under function Use.
  4. When package provides a handler usable inside another handler, it must be under function LocalUse.
  5. When function Init and/or Use can receive complex arguments, use an *Options structure pointer.
  6. When package provides a hanlder that binds a custom writer the response writer, call 
    defer c.Recover()
    before closing the binded custom writer.
